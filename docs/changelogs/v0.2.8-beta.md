# v0.2.8-beta — Closed Beta Package Test

## Added
- Live email signup and verification through the hosted Kodiak Auth backend.
- Matrix account creation wired into the v2 homeserver registration flow.
- Desktop updater validation using the hosted Kodiak update manifest.
- GIF/image, file/music, and video sharing flow restored for beta testing.
- Inline media cards for shared images, audio, video, and downloadable files.
- Safety Center report replies, report history, archive, and delete actions.
- User profile cards with message, friend, block, and report actions.
- Friend Center entry point for beta social testing.

## Changed
- Merged the staging backend/auth foundation into main.
- Updated packaged-client environment flow for hosted Matrix/Auth/API endpoints.
- Improved workspace shell layout, media message sizing, modal behavior, and profile popovers.
- Cleaned up Safety Center button behavior so it no longer inherits profile-pill styling.
- Reduced the stitched-panel feel by tightening workspace surface and layout rules.

## Fixed
- Fixed Matrix shared-secret registration by targeting the v2 Synapse listener.
- Fixed report moderation routes for reply, note, status, archive, and delete actions.
- Fixed stale local blob/avatar cache cleanup.
- Fixed several media-sharing messages rendering as plain placeholder text instead of usable media cards.
- Fixed several dev layout issues around top offsets, floating modals, and oversized panels.

## Beta Notes
- This is a closed beta build for trusted testers only.
- Android update testing is APK-over-APK for now and requires the same signing key.
- Desktop update testing uses the Kodiak hosted updater manifest.
- Some visual cleanup is still planned before wider testing.

## Tester Request
Install the older 0.2.7 package first, confirm login/signup and basic chat work, then update to 0.2.8. Please test signup, login, media sharing, Safety Center reports, profile actions, and update behavior.
