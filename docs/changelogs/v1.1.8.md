# Kodiak Connect v1.1.8

This update focuses on profile reliability, smoother chat navigation, and better desktop audio support.

## Profile reliability

- Improved profile persistence so saved display names, bios, and profile pictures are not replaced by empty fallback profiles after logging back in.
- Improved profile caching so the app can preserve known profile details if the profile service briefly returns default information.
- Improved backend profile storage handling to keep account/profile data stable across future updates.

## Chat navigation

- Improved chat scrolling so Kodiak Connect stays pinned to the latest message while you are already at the bottom.
- If you scroll up to read older messages, the app now respects that and does not force-jump you back down.
- Added a Latest jump button that appears once you are far enough up the chat, letting you quickly return to the newest message.

## Desktop sound support

- Added Linux-friendly Kodiak sound assets converted from the same original notification sounds.
- Improved desktop sound playback selection so Linux builds can use a more compatible audio format while keeping the same Kodiak sound design.
- Message sound settings and the sound test remain available in Settings.

## Desktop polish

- Continued improving desktop stability after the tray, settings, theme, and media handling updates from the previous release.
