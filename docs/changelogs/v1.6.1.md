# Kodiak Connect v1.6.1 Dev Update

Kodiak Connect v1.6.1 is a hotfix/stability update focused on mobile usability, Matrix media reliability, GIF previews, profile pictures, and packaged app updates.

## Fixed

- Fixed authenticated Matrix media rendering for uploaded images, screenshots, audio, video, and files.
- Restored GIF previews so Giphy shares render properly instead of showing raw fallback text.
- Added a safer Giphy fallback path when direct Matrix media upload is unavailable.
- Fixed profile picture saving by adding a fallback avatar path when Matrix media upload fails.
- Improved mobile layout behavior so the app is more usable on smaller screens.
- Improved access to the channel strip, chat history, message composer, media tools, profile/settings modals, and safety dialogs.

## Release Pipeline

- Confirmed the live web app and packaged apps are separate deployment paths.
- Confirmed packaged Windows, Linux, and Android apps require a versioned release because they use bundled frontend assets.
- Used the existing release workflow for Windows, Linux, and Android publishing.
- Fixed the Tauri CLI release build blocker.
- Fixed the VPS Linux build blocker caused by /opt/kodiak-connect repo permissions.

## Deployment Notes

- Web builds are synced to /var/www/kodiak-connect.
- Packaged update artifacts are published through updates.kodiak-connect.com.
- Desktop clients use the main updater manifest.
- Android clients use the Android latest manifest.

## Next

- Continue testing mobile layouts on real devices.
- Continue hardening Matrix upload/download behavior.
- Move Android builds from debug APKs to proper release APKs once signing is finalized.
