# Kodiak Connect 1.1.12

This update improves Kodiak Connect calling and mobile usability.

## What's new

- Added working voice call controls for direct messages.
- Added video call controls and camera on/off handling.
- Added clearer microphone and camera permission messages.
- Added call participant display with profile pictures.
- Added active speaker highlighting during calls.
- Improved mobile chat layout, composer placement, and member panel behavior.
- Improved media/link presentation in chat.
- Improved Android call permission support for microphone and camera.

## Notes

If you installed an older desktop build before the updater signing key was rotated, you may need to manually install this version once before automatic updates work normally again.
