# Kodiak Connect v1.1.7

This update focuses on making the desktop app feel smoother, more reliable, and easier to control across Windows and Linux.

## Desktop app improvements

- Added system tray support for Windows and Linux.
- Closing the desktop app now keeps Kodiak Connect running in the tray instead of fully quitting.
- Added a tray menu with Open, Toggle start minimized, and Quit.
- Added a Start minimized to tray setting under Themes.

## Messaging improvements

- Improved message sending responsiveness so you can start typing again immediately after sending.
- Reduced background refresh buildup that could make the Linux desktop app feel slower during longer chats.
- Improved received-message sound handling to avoid rapid sound spam.
- Added a sound test option in settings so desktop users can confirm message sounds work on their device.

## Settings and personalization

- Added Settings tabs for Profile, Sounds, and Themes.
- Added a Themes section with Default and System settings.
- System settings follows the device/browser-reported light or dark preference where supported.
- Moved sound controls into the desktop settings flow so packaged-app users do not need the web app to manage them.

## Files and media

- Added better drag-and-drop and paste handling for shared files where supported by the desktop webview.
- Improved attachment downloads so chat media can use a native save flow.
- Added clearer download/save status feedback.

## Interface fixes

- Fixed broken close-button symbols in settings, Friend Center, and Safety Center.
- Fixed title bar spacing so it no longer blocks the Kodiak icon.
- Improved compact desktop layouts so the chat stays usable at smaller window sizes.
- Restored member panel collapse behavior and cleaned up broken UI symbols.
- Kept the composer and attachment controls anchored to the chat area while scrolling.
