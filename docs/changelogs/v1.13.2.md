# Kodiak Connect v1.13.2

This update fixes voice-call UI behavior, improves direct message reliability, and stabilizes the mobile chat experience.

## What’s fixed

- Fixed an issue where a call could ring for the other user while the caller did not see the active call UI.
- Fixed call setup so the call panel appears immediately while the app prepares the microphone and connection.
- Fixed ended calls leaving the call/status panel stuck on screen.
- Fixed a mobile layout issue where call UI/status could interfere with the message composer.
- Fixed hidden direct messages not reopening reliably when a new message comes in.
- Improved the mobile chat composer so it stays anchored correctly at the bottom of the chat area.
- Added TURN relay support to improve call reliability across different networks.

## Notes

- Users should update to the latest version before testing calls.
- Windows installer signing/trust improvements are still being handled separately and are not part of this update.
