# Kodiak Connect v1.13.2

This update focuses on mobile usability and voice-call reliability.

## What’s improved

- Improved the mobile chat layout so conversations use the available screen space more cleanly.
- Refined the mobile message composer so typing and sending messages feels better on smaller screens.
- Improved voice-call connection reliability across more network conditions.
- Preserved the clean mobile sidebar and member-list drawer behavior while keeping the chat area usable.
- Continued polishing the app experience ahead of broader testing.

## Notes

- Users should update to the latest version before testing voice calls.
- Voice calls should now behave more reliably when users are on different networks.
- Windows installer trust/signing improvements are being handled separately and are not part of this update.
