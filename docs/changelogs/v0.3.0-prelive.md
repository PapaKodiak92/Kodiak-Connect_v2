# v0.3.0-prelive — Official Space Live Chat

## Added
- Fresh Matrix v2 backend.
- Real Matrix login against the v2 homeserver.
- Official Space server/channel shell.
- Matrix-backed `#general` chat.
- Read-only `#dev-updates` channel for curated development updates.
- Read-only `#announcements` channel for major official notices.
- First-time Official Space acknowledgement gate.
- Login rate-limit cooldown handling.
- Automated GitHub Actions changelog posting to `#dev-updates`.
- File-based changelog publishing from `docs/changelogs/*.md`.

## Changed
- Login now moves into the authenticated workspace shell.
- Official channels are separated from normal chat channels.
- Healthy Matrix connection labels are hidden from the product UI.
- Dev updates are now intended to be posted from curated changelog files instead of raw commit logs.

## Notes
- This is a pre-live milestone, not a production release.
- Changelogs should stay curated and user-facing.
- Raw commit spam should not be posted into user-visible channels.
