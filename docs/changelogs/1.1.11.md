# Kodiak Connect 1.1.11

This update improves mobile notification behavior and direct message reliability.

## What's improved

- Added Android push notification support for direct messages.
- Improved notification behavior so direct messages can notify you when the app is closed, backgrounded, or not focused on that conversation.
- Fixed closed direct message rooms so they can reappear when a new unread message arrives.
- Improved direct message unread behavior after a conversation has been hidden from the sidebar.
- Shortened friend presence timing so users move offline much faster after they close or leave the app.
- Improved friend online/offline notification timing for more realistic status updates.
- Improved the foundation for future call and mobile notification behavior.

## Notes

This release focuses on making mobile testing more reliable, especially for Android users testing direct messages, friend presence, and notification delivery.
