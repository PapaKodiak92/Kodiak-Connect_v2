# Kodiak Connect v1.1.9

This update is a Linux desktop audio hotfix.

## Linux desktop audio

- Added Linux audio runtime dependencies to the Debian package.
- Improved support for Kodiak Connect notification sounds on Linux desktop systems.
- Added support for common GStreamer/PipeWire audio components used by WebKitGTK-based desktop apps.
- Keeps the same Kodiak notification sound design from the previous release.

## Settings

- The sound test remains available in Settings so Linux users can confirm notification sounds after installing the update.
