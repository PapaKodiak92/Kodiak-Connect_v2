# Kodiak Connect v1.1.0 — Updater Validation Release

## Added
- Live desktop updater validation from the 1.0.0 baseline.
- Clean release artifact naming for Windows, Linux, and Android builds.
- Styled Kodiak Connect verification emails with official branding.
- Curated dev update posting flow for the `#dev-updates` channel.

## Changed
- Release automation now publishes through the hardened `kodiak` SSH account.
- Update manifests now publish through `updates.kodiak-connect.com`.
- Dev update automation can post directly from curated changelog files.

## Fixed
- Fixed release script paths after removing old v2 naming.
- Fixed update server root path for hosted manifests and artifacts.
- Fixed Linux `.deb` release publishing as part of the full release flow.

## Notes
- This update confirms that Kodiak Connect can move users from 1.0.0 to 1.1.0 through the hosted autoupdater.
