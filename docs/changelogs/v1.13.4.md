# Kodiak Connect v1.13.4

This hotfix focuses on voice-call stability, mobile calling, direct message recovery, and call UI cleanup.

## What’s fixed

- Fixed call UI behavior so the call panel appears correctly during direct message calls.
- Fixed call setup so the call panel appears while the app prepares the microphone and connection.
- Fixed ended calls leaving the call/status panel stuck on screen.
- Improved mobile voice-call permission handling.
- Improved mobile call layout so the call panel does not break the chat composer.
- Fixed hidden direct messages not reopening reliably when a new message comes in.
- Improved voice-call reliability across more devices and network conditions.

## Notes

- Users should update before testing calls again.
- Windows installer signing/trust improvements are still being handled separately and are not part of this update.
