# Kodiak Connect v1.1.10

This update fixes Linux desktop notification sound playback.

## Linux audio

- Improved Kodiak Connect notification sound playback on Linux desktop builds.
- Plays the same Kodiak sound files through a more reliable WebAudio path first.
- Keeps WAV and MP3 fallback support for desktop environments that handle audio differently.
- The sound test in Settings should now work on Linux systems where the previous audio path failed.
