# Kodiak Connect v1.1.5 — Mobile & Media Stability Patch

## Fixed
- Fixed uploaded images, screenshots, files, audio, and video not displaying correctly for other users.
- Fixed media previews that could appear as broken placeholders.
- Improved mobile layout so chat, channels, message input, media tools, and profile panels are easier to use on smaller screens.
- Improved attachment preview and download behavior across accounts.

## Notes
- This update focused on making Kodiak Connect more usable on mobile and more reliable when sharing media.
