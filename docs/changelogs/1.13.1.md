# Kodiak Connect 1.13.1

This update adds the first version of Kodiak Music Lounge and improves message formatting.

## What's new

- Added a dedicated Music Lounge experience to the Official Space.
- Added rotating music vibes like Random Hits, Dev Focus, Throwbacks, Rock, Rap, Country, and Chill.
- Added Spotify-friendly links so users can open the current vibe in their own Spotify app/account.
- Added local vibe voting controls for early Music Lounge feedback.
- Added multiline message composition.
- Added support for line breaks and cleaner formatting in announcement-style posts.
- Continued polishing the Kodiak Connect interface for community and development use.

## Notes

Music Lounge is starting as a shared vibe/listening hub. Deeper Spotify account linking and synced listening are planned for a future update.
