# Kodiak Connect v1.13.5

This minor debug patch improves voice-call behavior after live testing.

## What’s fixed

- Improved voice-call stability when a call temporarily disconnects or reconnects.
- Fixed remote audio attachment so users are less likely to appear as speaking while the other side cannot hear them.
- Improved call UI cleanup and restart behavior after a call ends.
- Improved mobile microphone/camera permission handling for packaged Android builds.
- Removed forced relay-only behavior so calls can use the best available connection path while still supporting TURN relay fallback.
- Continued polish for the direct message call panel and mobile chat layout.

## Notes

- Users should update before testing calls again.
- This is a debug-focused call reliability patch.
- Windows installer signing/trust improvements are still being handled separately and are not part of this update.
