
# Kodiak Connect v1.1.6 Dev Update



Kodiak Connect v1.1.6 is a stability release focused on mobile usability, Matrix media handling, GIF previews, profile pictures, and the packaged app release pipeline.



## Fixed



- Fixed Matrix media rendering so uploaded images, screenshots, audio, video, and files can be fetched through authenticated Matrix media requests.

- Restored GIF previews so Giphy shares render as media instead of plain raw links.

- Added a safer Giphy fallback path when direct Matrix media upload is unavailable.

- Fixed profile picture saving by adding a fallback avatar path when Matrix media upload fails.

- Improved mobile layout behavior so the app is more usable on small screens.

- Improved mobile access to the channel strip, chat history, message composer, media tools, profile/settings modals, and safety dialogs.



## Release Pipeline



- Confirmed the browser/web deploy and packaged app deploy paths are separate.

- Confirmed packaged Windows, Linux, and Android apps use bundled frontend assets and require a versioned release.

- Used the existing elease-all.ps1 workflow for Windows, Linux, and Android publishing.

- Fixed the local Tauri CLI release build blocker.

- Fixed the VPS Linux release blocker caused by /opt/kodiak-connect repo permissions.



## Deployment Notes



- The live web app is served from /var/www/kodiak-connect.

- The packaged app update artifacts are published through updates.kodiak-connect.com.

- Desktop clients use the main updater manifest.

- Android clients use the Android latest manifest.



## Next



- Continue testing mobile layouts on real devices.

- Continue hardening Matrix media upload/download behavior.

- Move Android release builds away from debug APKs when the signing/release path is ready.
