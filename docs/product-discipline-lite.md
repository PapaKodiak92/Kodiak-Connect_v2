# Kodiak Connect v2 Discipline

Kodiak Connect v2 is a clean rebuild.

## Rules

- v1 stays live until v2 is ready.
- v1 is reference only.
- Keep changes small and reviewable.
- Keep app composition thin.
- Keep feature logic under `src/features`.
- Keep platform logic under `src/platform`.
- Do not mix unrelated work in one patch.

## First product goal

Build a reliable private communication app before adding extra features.
