# Kodiak Connect Dev Update Template

Use this template for curated posts in `#dev-updates`.

Do not post raw commit logs. Keep updates clear, user-facing, and safe for public visibility.

## Template

```text
vX.Y.Z — Milestone Name

Added:
- Item one
- Item two

Changed:
- Item one

Fixed:
- Item one

Notes:
- Any testing notes or known limitations
```

## Example

```text
v0.3.0-prelive — Official Space Live Chat

Added:
- Official Space server/channel shell
- Matrix-backed #general chat
- Read-only #dev-updates channel
- Read-only #announcements channel
- First-time Official Space acknowledgement gate
- Login rate-limit cooldown handling

Changed:
- Login now moves into the authenticated workspace shell
- Official channels are separated from normal chat channels

Notes:
- This is still a pre-live milestone, not a production release
- Changelog posts should stay curated and manually approved
```
